初级教程

1. 基础语法

1.1 变量和数据类型

在 Luau 中，你可以使用 local 关键字声明变量。Luau 支持多种数据类型，包括数字、字符串、布尔值等。

-- 声明变量 local age = 18 local name = "Alice" local isStudent = true local score = 95.5 -- 打印变量 print("年龄: " .. age) print("姓名: " .. name) print("是否是学生: " .. tostring(isStudent)) print("分数: " .. score) -- 修改变量 age = 19 print("更新后的年龄: " .. age)

练习 1.1：变量和数据类型

声明几个变量，包括你的姓名、年龄、身高和是否喜欢编程，然后打印这些变量的值。

-- 声明变量 local myName = "Bob" local myAge = 20 local myHeight = 1.75 local likeProgramming = true -- 打印变量 print("我的姓名: " .. myName) print("我的年龄: " .. myAge) print("我的身高: " .. myHeight) print("我喜欢编程: " .. tostring(likeProgramming))

1.2 基本运算符

Luau 支持各种运算符，包括算术运算符、比较运算符和逻辑运算符。

-- 算术运算符 local a = 10 local b = 3 print("a + b = " .. (a + b)) -- 加法 print("a - b = " .. (a - b)) -- 减法 print("a * b = " .. (a * b)) -- 乘法 print("a / b = " .. (a / b)) -- 除法 print("a % b = " .. (a % b)) -- 取模 -- 比较运算符 print("a == b: " .. tostring(a == b)) -- 等于 print("a ~= b: " .. tostring(a ~= b)) -- 不等于 print("a > b: " .. tostring(a > b)) -- 大于 print("a < b: " .. tostring(a < b)) -- 小于 print("a >= b: " .. tostring(a >= b)) -- 大于等于 print("a <= b: " .. tostring(a <= b)) -- 小于等于 -- 逻辑运算符 local c = true local d = false print("c and d: " .. tostring(c and d)) -- 逻辑与 print("c or d: " .. tostring(c or d)) -- 逻辑或 print("not c: " .. tostring(not c)) -- 逻辑非

练习 1.2：基本运算符

创建两个变量，分别存储两个数字，然后使用各种运算符计算并打印结果。

-- 创建两个变量 local num1 = 20 local num2 = 5 -- 算术运算 print("num1 + num2 = " .. (num1 + num2)) print("num1 - num2 = " .. (num1 - num2)) print("num1 * num2 = " .. (num1 * num2)) print("num1 / num2 = " .. (num1 / num2)) print("num1 % num2 = " .. (num1 % num2)) -- 比较运算 print("num1 == num2: " .. tostring(num1 == num2)) print("num1 > num2: " .. tostring(num1 > num2)) print("num1 < num2: " .. tostring(num1 < num2))

1.3 条件语句

条件语句允许你根据条件执行不同的代码块。

-- if 语句 local score = 85 if score >= 90 then print("优秀") elif score >= 80 then print("良好") elif score >= 60 then print("及格") else print("不及格") end -- 嵌套 if 语句 local age = 16 local hasID = false if age >= 18 then print("可以进入") else if hasID then print("有身份证，可以进入") else print("不能进入") end end

练习 1.3：条件语句

创建一个变量存储一个数字，然后使用条件语句判断这个数字是正数、负数还是零。

-- 创建一个变量 local number = -5 -- 判断数字的类型 if number > 0 then print("这是一个正数") elif number < 0 then print("这是一个负数") else print("这是零") end

1.4 循环

循环允许你重复执行代码块。

-- for 循环 print("1 到 5 的数字:") for i = 1, 5 do print(i) end -- 带有步长的 for 循环 print("1 到 10 的偶数:") for i = 2, 10, 2 do print(i) end -- while 循环 local count = 1 print("使用 while 循环:") while count <= 5 do print(count) count = count + 1 end -- repeat-until 循环 local num = 1 print("使用 repeat-until 循环:") repeat print(num) num = num + 1 until num > 5

练习 1.4：循环

使用循环计算 1 到 100 的和。

-- 使用 for 循环计算 1 到 100 的和 local sum = 0 for i = 1, 100 do sum = sum + i end print("1 到 100 的和: " .. sum) -- 使用 while 循环计算 1 到 100 的和 local sum2 = 0 local i = 1 while i <= 100 do sum2 = sum2 + i i = i + 1 end print("1 到 100 的和 (while): " .. sum2)

2. 函数

2.1 函数定义和调用

函数是可重用的代码块，用于执行特定的任务。

-- 定义函数 function greet(name) print("你好, " .. name .. "!") end -- 调用函数 greet("Alice") greet("Bob") -- 带返回值的函数 function add(a, b) return a + b end local result = add(5, 3) print("5 + 3 = " .. result) -- 多个返回值 function getInfo() return "Alice", 25 end local name, age = getInfo() print("姓名: " .. name) print("年龄: " .. age)

练习 2.1：函数定义和调用

创建一个函数，计算两个数字的乘积，并返回结果。然后调用这个函数并打印结果。

-- 定义函数 function multiply(a, b) return a * b end -- 调用函数 local result = multiply(6, 7) print("6 * 7 = " .. result) -- 测试不同的参数 print("3 * 4 = " .. multiply(3, 4)) print("5 * 10 = " .. multiply(5, 10))

2.2 函数参数

函数可以接受多个参数，也可以有默认参数值。

-- 多个参数 function calculateArea(length, width) return length * width end print("长方形面积: " .. calculateArea(5, 3)) -- 默认参数 function sayHello(name, greeting) greeting = greeting or "你好" print(greeting .. ", " .. name .. "!") end sayHello("Alice") -- 使用默认问候语 sayHello("Bob", "早上好") -- 使用自定义问候语 -- 可变参数 function sum(...) local total = 0 for _, num in ipairs({...}) do total = total + num end return total end print("1 + 2 + 3 = " .. sum(1, 2, 3)) print("4 + 5 + 6 + 7 = " .. sum(4, 5, 6, 7))

练习 2.2：函数参数

创建一个函数，计算一个数字的阶乘（例如，5! = 5 × 4 × 3 × 2 × 1 = 120）。

-- 计算阶乘的函数 function factorial(n) if n <= 1 then return 1 else return n * factorial(n - 1) end end -- 测试函数 print("5! = " .. factorial(5)) print("3! = " .. factorial(3)) print("10! = " .. factorial(10))

3. 表（Tables）基础

3.1 表的创建和访问

表是 Luau 中最基本的数据结构，可以存储多个值。

-- 创建表 local fruits = {"苹果", "香蕉", "橙子"} -- 访问表元素（索引从 1 开始） print("第一个水果: " .. fruits[1]) print("第二个水果: " .. fruits[2]) -- 修改表元素 fruits[2] = "葡萄" print("修改后的第二个水果: " .. fruits[2]) -- 添加新元素 fruits[4] = "西瓜" print("第四个水果: " .. fruits[4]) -- 获取表长度 print("水果数量: " .. #fruits)

练习 3.1：表的创建和访问

创建一个包含你喜欢的电影的表，然后访问、修改和添加元素。

-- 创建电影表 local movies = {"肖申克的救赎", "泰坦尼克号", "盗梦空间"} -- 访问表元素 print("第一部电影: " .. movies[1]) print("第二部电影: " .. movies[2]) print("第三部电影: " .. movies[3]) -- 修改表元素 movies[2] = "阿凡达" print("修改后的第二部电影: " .. movies[2]) -- 添加新元素 movies[4] = "星际穿越" movies[5] = "蝙蝠侠:黑暗骑士" -- 打印所有电影 print("所有电影:") for i, movie in ipairs(movies) do print(i .. ". " .. movie) end -- 打印电影数量 print("电影数量: " .. #movies)

3.2 字典表

表也可以用作字典，使用键值对存储数据。

-- 创建字典表 local person = { name = "Alice", age = 25, city = "New York" } -- 访问字典元素 print("姓名: " .. person.name) print("年龄: " .. person.age) print("城市: " .. person.city) -- 修改字典元素 person.age = 26 print("更新后的年龄: " .. person.age) -- 添加新元素 person.job = "工程师" print("职业: " .. person.job) -- 遍历字典 print("所有信息:") for key, value in pairs(person) do print(key .. ": " .. tostring(value)) end

练习 3.2：字典表

创建一个字典表，存储一个学生的信息，包括姓名、年龄、班级和成绩，然后遍历并打印所有信息。

-- 创建学生信息字典 local student = { name = "Bob", age = 16, class = "高一(1)班", grades = { math = 90, english = 85, science = 95 } } -- 访问基本信息 print("姓名: " .. student.name) print("年龄: " .. student.age) print("班级: " .. student.class) -- 访问成绩 print("数学成绩: " .. student.grades.math) print("英语成绩: " .. student.grades.english) print("科学成绩: " .. student.grades.science) -- 遍历所有信息 print("\n所有学生信息:") for key, value in pairs(student) do if type(value) == "table" then print(key .. ":") for subject, grade in pairs(value) do print(" " .. subject .. ": " .. grade) end else print(key .. ": " .. tostring(value)) end end

4. 基本输入输出

4.1 输出

使用 print 函数可以在控制台输出信息。

-- 输出字符串 print("Hello, Luau!") -- 输出变量 local name = "Alice" print("我的名字是 " .. name) -- 输出多个值 local age = 25 print("姓名:", name, "年龄:", age) -- 输出表达式结果 print("5 + 3 = " .. (5 + 3))

4.2 注释

注释是代码中不会被执行的部分，用于解释代码的功能。

-- 这是单行注释 --[[ 这是 多行注释 可以跨越多行 ]] -- 注释可以放在代码旁边 local age = 18 -- 存储年龄 -- 注释也可以用于暂时禁用代码 -- print("这段代码不会执行") print("这段代码会执行")
