中级教程

1. 中级语法概念

1.1 表（Tables）

表是 Luau 中最强大的数据结构，它可以用作数组、字典、对象等。

-- 创建一个表 local person = { name = "Alice", age = 25, hobbies = {"reading", "coding", "hiking"} } -- 访问表中的元素 print(person.name) -- 输出: Alice print(person.hobbies[1]) -- 输出: reading -- 修改表中的元素 person.age = 26 -- 添加新元素 person.city = "New York" -- 遍历表 for key, value in pairs(person) do if type(value) == "table" then print(key .. ":") for i, hobby in ipairs(value) do print(" " .. i .. ": " .. hobby) end else print(key .. ": " .. tostring(value)) end end

练习 1.1：创建和操作表

创建一个包含学生信息的表，包括姓名、年龄、成绩（数学、英语、科学），然后计算并添加平均成绩，最后遍历输出所有信息。

-- 创建学生信息表 local student = { name = "Bob", age = 16, grades = { math = 90, english = 85, science = 95 } } -- 计算平均成绩 local total = 0 local count = 0 for subject, grade in pairs(student.grades) do total = total + grade count = count + 1 end student.average = total / count -- 遍历输出所有信息 print("学生信息：") print("姓名: " .. student.name) print("年龄: " .. student.age) print("成绩:") for subject, grade in pairs(student.grades) do print(" " .. subject .. ": " .. grade) end print("平均成绩: " .. student.average)

1.2 函数高级特性

Luau 支持函数作为参数传递、返回值，以及匿名函数等高级特性。

-- 函数作为参数 function applyOperation(num1, num2, operation) return operation(num1, num2) end -- 匿名函数 local add = function(a, b) return a + b end local multiply = function(a, b) return a * b end print(applyOperation(5, 3, add)) -- 输出: 8 print(applyOperation(5, 3, multiply)) -- 输出: 15 -- 闭包 function createCounter() local count = 0 return function() count = count + 1 return count end end local counter = createCounter() print(counter()) -- 输出: 1 print(counter()) -- 输出: 2 print(counter()) -- 输出: 3

练习 1.2：使用函数高级特性

创建一个函数，接受一个数字列表和一个操作函数，返回应用操作后的新列表。例如，传入列表 [1, 2, 3] 和函数 function(x) return x * 2 end，应返回 [2, 4, 6]。

-- 应用操作到列表 function applyToList(list, operation) local result = {} for i, value in ipairs(list) do result[i] = operation(value) end return result end -- 测试 local numbers = {1, 2, 3, 4, 5} -- 乘以 2 local doubled = applyToList(numbers, function(x) return x * 2 end) -- 计算平方 local squared = applyToList(numbers, function(x) return x * x end) -- 输出结果 print("原始列表:") for i, num in ipairs(numbers) do print(num) end print("乘以 2:") for i, num in ipairs(doubled) do print(num) end print("平方:") for i, num in ipairs(squared) do print(num) end

1.3 模块系统

Luau 支持模块系统，可以将代码组织成可重用的模块。

-- 模块文件: mathUtils.lua local MathUtils = {} function MathUtils.add(a, b) return a + b end function MathUtils.subtract(a, b) return a - b end function MathUtils.multiply(a, b) return a * b end function MathUtils.divide(a, b) if b == 0 then error("除数不能为零") end return a / b end return MathUtils -- 使用模块 local MathUtils = require("mathUtils") print(MathUtils.add(5, 3)) -- 输出: 8 print(MathUtils.subtract(5, 3)) -- 输出: 2 print(MathUtils.multiply(5, 3)) -- 输出: 15 print(MathUtils.divide(6, 3)) -- 输出: 2

练习 1.3：创建和使用模块

创建一个字符串工具模块，包含反转字符串、计算字符串长度、检查字符串是否为空等函数，然后在主代码中使用该模块。

-- 模块文件: stringUtils.lua local StringUtils = {} function StringUtils.reverse(str) local reversed = "" for i = #str, 1, -1 do reversed = reversed .. string.sub(str, i, i) end return reversed end function StringUtils.length(str) return #str end function StringUtils.isEmpty(str) return str == "" end function StringUtils.trim(str) return string.gsub(str, "^%s*(.-)%s*$", "%1") end return StringUtils -- 使用模块 local StringUtils = require("stringUtils") local testString = " Hello, Luau! " print("原始字符串: '" .. testString .. "'") print("反转字符串: '" .. StringUtils.reverse(testString) .. "'") print("字符串长度: " .. StringUtils.length(testString)) print("是否为空: " .. tostring(StringUtils.isEmpty(testString))) print("去除空白: '" .. StringUtils.trim(testString) .. "'")

2. 实用编程技巧

2.1 错误处理

Luau 提供了错误处理机制，可以捕获和处理运行时错误。

-- 基本错误处理 try { -- 可能出错的代码 local result = 10 / 0 print(result) } catch err { -- 错误处理 print("错误: " .. tostring(err)) } -- 自定义错误 function divide(a, b) if b == 0 then error("除数不能为零") end return a / b end try { local result = divide(10, 0) print(result) } catch err { print("捕获到错误: " .. tostring(err)) }

练习 2.1：错误处理练习

创建一个函数，接受一个数字列表，计算列表中所有数字的平均值。如果列表为空，抛出错误。使用错误处理机制捕获并处理这个错误。

function calculateAverage(numbers) if #numbers == 0 then error("列表不能为空") end local sum = 0 for _, num in ipairs(numbers) do sum = sum + num end return sum / #numbers end -- 测试 local testCases = { {1, 2, 3, 4, 5}, {}, {10, 20, 30} } for i, testCase in ipairs(testCases) do try { local average = calculateAverage(testCase) print("测试用例 " .. i .. " 平均值: " .. average) } catch err { print("测试用例 " .. i .. " 错误: " .. tostring(err)) } end

练习 2.2：性能优化练习

创建一个函数，生成斐波那契数列的前 n 项，使用记忆化技术提高性能。

-- 使用记忆化技术优化斐波那契数列 local memo = {} function fibonacci(n) if n <= 1 then return n end if memo[n] then return memo[n] end memo[n] = fibonacci(n - 1) + fibonacci(n - 2) return memo[n] end -- 测试 local start = os.clock() for i = 1, 40 do print("fibonacci(" .. i .. ") = " .. fibonacci(i)) end local finish = os.clock() print("执行时间: " .. (finish - start) .. " 秒")

2.3 代码组织

良好的代码组织可以提高代码的可读性和可维护性。

-- 按功能组织代码 local GameManager = {} -- 私有变量 local players = {} local gameStarted = false -- 私有函数 local function validatePlayer(player) return player and player.name and player.level end -- 公共方法 function GameManager.addPlayer(player) if validatePlayer(player) then table.insert(players, player) return true end return false end function GameManager.startGame() if #players >= 2 then gameStarted = true print("游戏开始！") return true end print("玩家不足，无法开始游戏") return false end function GameManager.getPlayers() return players end return GameManager

练习 2.3：代码组织练习

创建一个简单的库存管理系统，包含添加物品、移除物品、查看库存等功能，使用良好的代码组织方式。

local InventorySystem = {} -- 私有变量 local inventory = {} -- 私有函数 local function findItem(itemName) for i, item in ipairs(inventory) do if item.name == itemName then return i, item end end return nil end -- 公共方法 function InventorySystem.addItem(itemName, quantity) local index, existingItem = findItem(itemName) if existingItem then existingItem.quantity = existingItem.quantity + quantity else table.insert(inventory, {name = itemName, quantity = quantity}) end print("添加了 " .. quantity .. " 个 " .. itemName) end function InventorySystem.removeItem(itemName, quantity) local index, existingItem = findItem(itemName) if existingItem then if existingItem.quantity >= quantity then existingItem.quantity = existingItem.quantity - quantity if existingItem.quantity == 0 then table.remove(inventory, index) end print("移除了 " .. quantity .. " 个 " .. itemName) return true else print("库存不足") return false end else print("物品不存在") return false end end function InventorySystem.viewInventory() if #inventory == 0 then print("库存为空") return end print("当前库存:") for _, item in ipairs(inventory) do print("- " .. item.name .. ": " .. item.quantity) end end return InventorySystem -- 使用示例 local Inventory = require("InventorySystem") Inventory.addItem("苹果", 5) Inventory.addItem("香蕉", 3) Inventory.viewInventory() Inventory.removeItem("苹果", 2) Inventory.viewInventory() Inventory.removeItem("香蕉", 5)

3. 常见问题解决方案

3.1 表的深拷贝

在 Luau 中，表是引用类型，直接赋值会共享引用。如果需要创建表的深拷贝，可以使用以下方法。

function deepCopy(original) local copy = {} for key, value in pairs(original) do if type(value) == "table" then copy[key] = deepCopy(value) else copy[key] = value end end return copy end -- 测试 local original = { name = "Alice", age = 25, address = { street = "123 Main St", city = "New York" } } local copy = deepCopy(original) -- 修改原始表 original.address.city = "Los Angeles" -- 查看结果 print("原始表城市: " .. original.address.city) -- 输出: Los Angeles print("拷贝表城市: " .. copy.address.city) -- 输出: New York

3.2 字符串处理

Luau 提供了丰富的字符串处理函数，可以用于各种字符串操作。

-- 字符串分割 function splitString(str, delimiter) local result = {} local pattern = "([^" .. delimiter .. "]+)" for match in string.gmatch(str, pattern) do table.insert(result, match) end return result end -- 测试 local str = "apple,banana,orange" local fruits = splitString(str, ",") for i, fruit in ipairs(fruits) do print(fruit) end -- 字符串替换 local text = "Hello, World!" local replaced = string.gsub(text, "World", "Luau") print(replaced) -- 输出: Hello, Luau! -- 字符串查找 local position = string.find(text, "World") print("World 开始于位置: " .. position) -- 输出: 8

3.3 时间处理

Luau 提供了时间处理函数，可以用于获取当前时间、格式化时间等。

-- 获取当前时间 local currentTime = os.time() print("当前时间戳: " .. currentTime) -- 格式化时间 local formattedTime = os.date("%Y-%m-%d %H:%M:%S", currentTime) print("格式化时间: " .. formattedTime) -- 计算时间差 local startTime = os.time() -- 模拟一些操作 for i = 1, 1000000 do local x = i * i end local endTime = os.time() local elapsed = endTime - startTime print("执行时间: " .. elapsed .. " 秒")
